## 3. ЭКРАН — СОЗДАНИЕ ТЕМЫ (scrForumCreate)

### Хедер
```
[×]    Новая тема    [Опубликовать]
```
- Кнопка «Опубликовать» — disabled (opacity 0.35) пока не заполнено
- Активируется: категория + заголовок >= 5 симв + текст >= 10 симв

### Форма
1. **Категория** — кнопка → bottom sheet с 7 категориями + цветными точками
2. **Заголовок** — input, max 120 симв, счётчик 0/120
3. **Описание** — textarea, max 3000 симв, счётчик 0/3000
4. **XP-подсказка** — «Создание темы: +10 XP»
5. **Превью** — появляется когда заполнен заголовок

### Доступ
- Только пользователи с платной подпиской (plan != 'free' и plan != '')
- FAB скрыт у free-пользователей
- При попытке создать без подписки → тост «Доступно с подпиской»
